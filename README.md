# Python para Data Science: Primeiros Passos

Repositório com os desafios realizados no curso **Python para Data Science: Primeiros Passos** da [Alura](https://www.alura.com.br).

## Sobre
Cada pasta representa uma aula do curso.  
Aqui estão os desafios propostos, feitos com o objetivo de praticar lógica e aplicar os conceitos de Python voltados a Data Science.

## Tecnologias utilizadas
- Python 3
- Google Colab
- VS Code

## Observação
O foco deste repositório é **documentar o progresso** nos estudos, não compartilhar soluções completas de desafios.
